## ЁЯМЯ What is LuminaryJr?

LuminaryJr is an interactive, browser-based photo editor built with the magical powers of **React**, **Vite**, and **TypeScript**. Add filters, draw on your photos, tweak the brightness, or just make things look totally wild!

We built this so that editing photos feels like playing a game. Try zooming around, applying epic overlays, or letting the AI do something totally unexpected! ЁЯдЦтЬи

## ЁЯЪА Features

- ЁЯОн **Instant Filters:** Turn a boring photo into a masterpiece with a single click.
- ЁЯОи **Draw Anything:** Grab the brush tool and doodle your heart out.
- тЬи **Sweet Adjustments:** Tweak brightness, contrast, and saturation like a pro.
- ЁЯНО **Liquid Glass UI:** Experience the ultimate smooth, interactive Apple-style liquid glass theme switcher.
- ЁЯУе **Export to Save:** Proud of your creation? Download it instantly and show your friends!

## ЁЯТ╗ Tech Stack

- **Framework:** React 19
- **Build Tool:** Vite
- **Language:** TypeScript
- **Styling:** Tailwind CSS + Custom Animations (Framer Motion)
- **State Management:** Zustand
- **Canvas Power:** Fabric.js

## ЁЯЫая╕П Let's Build It! (Local Setup)

Want to run LuminaryJr on your own computer? It's super easy! 

1. **Clone the repo:**
   ```bash
   git clone https://github.com/kushalchalla981-tech/luminaryjr.git
   cd luminaryjr
   ```

2. **Install the magic potions (dependencies):**
   ```bash
   npm install
   ```

3. **Start the engine:**
   ```bash
   npm run dev
   ```

4. **Open your browser** and head over to `http://localhost:5173`. Boom! You're an editor now. ЁЯОЙ
